Parse-burp.rb
=============
Is a simple ruby script for parsing the XML generated output from Burp
Scanner results.

Contact: royce.e.davis@gmail.com

Updated 09/09/2013
------------------
    $ ./parse-burp.rb 
    Parse Burpsuite XML output into Tab delimited results
    Example: ./parse-brup.rb <xml-file> > output.csv
